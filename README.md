# HQ-fragANDreco

Heavy-quark (HQ) hadronization framework in vacuum and QGP.

## Features

- **Vacuum fragmentation**
  - HQ → hadrons using HQET-inspired fragmentation  
  - Based on:  
    [Charm-hadron production in pp and AA collisions](https://arxiv.org/abs/2002.00392)

- **Hybrid hadronization in QGP**
  - Fragmentation + recombination (coalescence)  
  - Based on:  
    [Charmed hadron chemistry in relativistic heavy-ion collisions](https://arxiv.org/abs/1911.00456)
